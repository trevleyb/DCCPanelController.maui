# Tiles

The DCC Panel Controller supports a number of different tiles that can be added to a panel. These tiles are used to represent the various entities on your layout such as tracks, turnouts, signals, etc. Each tile can be placed anywhere on the panel, and most (other than track tiles) can be resized. All tiles upport properties and configuration and you can select many tiles and set the properties for all at the same time. 

This page will provide guidance on what each tile type is, what it is for and what properties it supports.

Note that in the palette panel on the panel editor you will see a group called **Mainline** and a group called **Branchline**. These are the same tiles with the only difference being that the Branchline tiles have the tile type set to branchline. You can add a normal mainline tile and change its type to branchline which is te same as dragging a branchline tile to the panel. 

Adding panels can be done in 1 of 3 ways:

1. You can click or tap on a tile in the tile palette and it will be marked as selected and then you can click or tap on the panel and that tile will be placed into that panel position. 
2. You can drag a tile from the palette to the panel. 
3. You can switch to copy mode and drag any tile already on the panel and it will copy to wherever you release your drag.
